Checks if the package the dependency points to is in the correct version restricted by the dependency.

For example, in the new package:
<test_depend version_lte="1.0.0">catkin</depend>
and in catkin we have:
<version>1.1.0</version>
This means that compiling the new package will fail because it requires a version at may be at most 1.0.0, but the available package is 1.1.0, higher than the imposed maximum.

This inspection offers two ways to fix this:

  1. change the dependency so that it includes the current version
  2. Remove the dependency tag(s).
